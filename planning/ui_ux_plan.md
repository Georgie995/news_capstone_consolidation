# UI/UX plan (front end)

## Pages (templates)
- Home page: entry point/navigation
- Login page
- Article list page
- Article detail page
- Editor review page (pending articles)

## UX principles used
- Clear navigation links
- Role-based visibility (users see actions they are permitted to take)
- Simple forms for create/update flows
- Confirmation step for deletes

## Accessibility considerations (basic)
- Use readable text contrast and clear headings
- Avoid overloading pages with dense text
